# INITIATING ALEMBIC ENVIRONMENTS

## Initiate new files
After installing, run the following command in your terminal at the project root.
```
alembic init alembic
```

This will create a new folder named `/alembic` and an ini `alembic.ini` file in your 
project root, containing the following files / directories.

| File/Directory           | Description                                                                                                                                                                                   |
|--------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `alembic/versions`       | This directory keeps `.py` files that can upgrade/downgrade your schema as well as providing recorded version ID. In other words, each `.py` file generated here is a version of your schema. |
| `alembic/env.py`         | Configure your project with `alembic`, including but not limited to: `--autogenerate` base, importing models, etc.                                                                            |
| `alembic/README`         | Not sure yet. Probably a landing script.                                                                                                                                                      |
| `alembic/script.py.mako` | This is the template file for autogenerated version files. Mostly do not need to be changed.                                                                                                  |
| `alembic.ini`            | Static configurations, think of this as meta-configuration.                                                                                                                                   |

## Configure `alembic.ini`
Type in the following, this is the setup for autogenerated file slugs as well as 
indicating the name of the alembic folder.
```
[alembic]
script_location = alembic
file_template = %%(year)d-%%(month).2d-%%(day).2d_%%(slug)s
```
